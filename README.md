## Comunica

Painel de monitoramento de projetos.
Com o painel, você pode: 
- Cadastrar domínios e monitorar status.
- Receber logs de erros.
- Receber status de backups.


## Dependências Composer

- Laravel Audit
- Laravel Permission
- Laravel JetStream
- Laravel Backup
- Laravel Reverb
- Laravel Horizon
